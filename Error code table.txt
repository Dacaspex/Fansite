Error code table


== OLD ==

0 - No error
1 - Successfully registered
2 - Something went wrong while registereing
3 - Successfully logged in
4 - Blog id didn't parse correctly
5 - Logged out successfully
6 - Something went wrong while checking your login status
7 - You can't register while logged in


== NEW ==

0 - no error
1 - Succesfully registered (success, redirect -> index)
2 - Something went wrong while registered (warning, redirect -> index)
3 - Your password and the password check don't match (warning)
4 - The username is not available (warning)
5 - The format of your username, password and or email adress are wrong. You can only use letters and numbers in your username and password (warning)
